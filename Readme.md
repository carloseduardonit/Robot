## Configuração de variaveis

- **Verificar a  versão do Python:**

``` dos
    python -v
```

- **Verificar  biblioteca instalada:**

``` dos
    pip freeze
```

- **Instalar Robot Framework e Verificar a versão do Robot Framework:**

``` dos
    pip install robotframework
    robot --version
```
